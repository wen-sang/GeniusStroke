# 变更记录

## v2.12.1-public.1

- 修复理杏仁指数日行情旧 `source_code` 导致接口返回空数据的问题。
- 新增理杏仁指数日行情运行时 `source_code` 兜底，避免历史旧路由阻断同步。
- 更新公开版代码基线至 v2.12.1。

## v2.12.0-public.1

- 新增账户绩效指标页，展示单位净值、累计收益率、年化收益率、最大回撤、年化波动率和交易质量。
- 新增账户绩效只读接口，支持按当前有效绩效段返回绩效统计结果。
- 优化账户历史净值计算，支持出入金、分红、拆分等场景下的份额法净值和资金加权收益。
- 修复复杂出入金场景下 XIRR 可能不显示的问题。
- 更新公开版代码基线至 v2.12.0。

## v2.11.3-public.1

- 修复买入交易输入标的代码时未读取内部基础档案的问题，标的识别改为仅使用系统内基础档案。
- 买入提交增加内部档案存在性校验，避免录入未维护基础档案的标的代码。
- 更新公开版代码基线至 v2.11.3。

## v2.11.2-public.1

- 修复理杏仁目录全量同步在目录项较多时可能失败的问题。
- 优化理杏仁分页目录拉取，按接口总数与成功空页共同判断分页结束。
- 更新公开版代码基线至 v2.11.2。

## v2.11.1-public.1

- 修复理杏仁股票目录在接口未返回 `listingStatus` 时无法进入统一搜索的问题，改用 `ipoDate` 判断已上市股票。
- 更新公开版代码基线至 v2.11.1。

## v2.11.0-public.1

- 新增 A 股股票数据源接入，支持通过统一目录搜索新增股票档案。
- 新增理杏仁 A 股股票日 K 采集路由，股票行情按不复权口径写入标准行情表。
- 补全 TickFlow 股票日 K 采集验证，并统一 A 股成交量单位，将接口返回的“手”转换为系统标准“股”。
- 优化目录搜索结果，已添加标的可见但不可重复选择。
- 更新公开版代码基线至 v2.11.0。

## v2.10.2-public.1

- 修复 LOF 标的每日行情采集接口路由，LOF 与 ETF 统一使用理杏仁基金日行情接口，指数行情接口保持不变。
- 更新公开版代码基线至 v2.10.2。

## v2.10.1-public.1

- 修复基金净值同步中交易所参数未传递导致批量更新失败的问题。
- 更新公开版代码基线至 v2.10.1。

## v2.9.0

- 精简持仓管理行情列，移除 `振幅`、`涨跌额`、`换手率` 展示。
- 实时行情接口不再返回 `amplitude`、`change_amt`、`turnover` 字段，保持持仓页前后端字段一致。
- 更新行情刷新和持仓列表回归测试，覆盖移除字段后的接口与 UI 合同。

## v2.7.1

- 升级数据更新入口交互：移除旧横幅与抽屉面板，改为导航按钮状态与轻量浮层卡片。
- 优化数据更新状态展示，区分运行中、成功、部分失败和失败，并集中展示失败标的与错误详情。
- 调整前端 SSE 生命周期，避免关闭浮层时中断运行中的任务；数据更新完成后不再触发页面自动刷新。
- 保持 `/api/data-sync/trigger` 兼容无请求体调用，并接受 `retry` 请求体字段，为前台重试交互预留合同。
- 迁移数据更新相关前端、CSS、DOM 与 Playwright 测试合同，避免继续绑定旧 UI。

## v2.7.0

- 行情列表新增近 22 日、近 60 日、近 6 个月、近一年涨幅字段，并支持升序 / 降序排序。
- 新增 `dat_market_return_snapshot` 快照表，行情采集完成后自动计算区间涨幅，接口查询不再动态扫描历史行情。
- 优化行情页交互：交易品种隐藏基本面入口，行情表头排序箭头和涨跌颜色按视觉规范展示。

## v2.6.1

- 行情UI调整：优化基础档案表单布局，移除冗余的代码输入框提示并完成表单输入框绝对垂直对称对齐，清理无用DOM引用与死代码。

## v2.6.0

- 升级行情 Tab 交互，支持交易品种 / 指数品牌切换和行情 / 技术 / 基本面视图切换。
- 重构管理档案弹窗，新增列表 / 新增 / 编辑视图与理杏仁、TickFlow 标的目录查询回填。
- 新增 `/api/v1/catalog/search` 统一目录搜索接口，保留旧档案搜索接口兼容。
- 补齐行情、指标和基本面接口改造前 / 改造后 P95 对比，关键接口无 5xx 且满足 105% 门槛。

## v2.5.0-public.2

- 新增外部标的目录能力，支持在新增档案时从 TickFlow / 理杏仁目录搜索并快速填充标的信息。
- 增加外部标的目录缓存表与同步日志表，公开版保留对应 Alembic 迁移链。
- 合入生产基线更新，包含前端模块化入口、数据质量能力、交易日历和手工订单幂等修复。
- 更新公开版文档为中文说明。
